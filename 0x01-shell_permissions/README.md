<h1>Shell Permissions</h1>
<h2>Command scripts to execute permissions in shell</h2>
0-iam_betty - This script switches the current user to the user betty
<hr>
1-who_am_i - This script prints the effetive username of the current user
<hr>
2-groups - This script prints all the groups the current user is a part of.
<hr>
3-new_owner - This script changes the owner of the file hello to the user betty.
<hr>
4-empty - This script creates an empty file called hello 
<hr>
5-execute - This script adds execute permission to the owner of the file hello
<hr>
6-multiple_permissions - This script adds execute permssion to the owner and the group owner, and read permission to the users, to the file hello.
<hr>
